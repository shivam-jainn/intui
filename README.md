# Intui — LeetCode‑style coding playground (portfolio)

A compact showcase of a LeetCode-style coding playground — built as a personal portfolio project to demonstrate full‑stack skills ,devops and system design choices.

---

## Short project highlights

- In-browser editor with language switch (Python / C++).
- Execution server compiles/runs test drivers and returns structured results on Google Cloud Run.
- Questions list, problem view, and submission history (DB-backed).
- Focused UX designed for interview walkthroughs and demos.

---

## Tech stack

- Frontend: `Next.js`, `React`, `TypeScript`, `Mantine`, `CodeMirror`
- Backend / infra: `Go` execserver, `Prisma` + `Postgres`, Docker
- Tooling: `Storybook`, `Jest`, `pnpm`
- Infra : `Terraform` , `Google Cloud Run`
